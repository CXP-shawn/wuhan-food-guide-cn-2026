# 筛选方法与数据来源

## 一、数据源与优先级

1. **大众点评必吃榜 / 高分榜 / 口味榜**：武汉分站公开榜单页面
2. **Google Maps 武汉三区高分餐厅**：通过 Apify `google_maps` scraper 抓取，过滤条件 rating ≥ 4.0 且为餐厅类目
3. **小红书 / 知乎 / 公众号**：武汉本地美食博主自然推荐（非软文）
4. **Perplexity 深度检索**：武汉三镇公开美食报道、2024-2025 年必吃榜
5. **本地老饕共识**：老字号、开业 5 年以上、多个独立来源自发提及

本批数据抓取与研究时段：2026-04-22。

## 二、广告过滤规则

入选前每家候选店都经过以下筛查：

- **真实评论占比**：如果评论区大量是"探店体验 / 霸王餐 / 试吃招募 / 送礼"标签，淘汰
- **自发口碑 vs 推广**：需要至少 2 个独立来源自发提及，且评价有具体菜品描述而非空洞好评
- **连锁快餐排除**：肯德基、麦当劳、必胜客、星巴克、瑞幸、德克士、汉堡王等一律不收录（除非是武汉本地特色门店，如某些历史百货楼层的老连锁）
- **新店淘汰**：新开业 3 个月内、评价数 < 30 的店铺数据不足，不收录
- **话题/网红店**：仅靠打卡流量、菜品口碑差的店不收录

## 三、重辣过滤规则

- **排除**：重庆火锅店、正宗湘菜馆、辣子鸡/水煮鱼等重辣专门店
- **保留微辣**：湖北本地菜里常见的微辣家常做法（如藕带、莲藕排骨、清蒸鱼类）
- **保留武汉经典**：热干面、鸭脖、豆皮、烧麦、汤包、糊汤粉、牛肉面、面窝等武汉本地小吃全部保留，并在辣度说明里标注是否可要求不辣/微辣版本
- **混合型店**：如果一家店既有重辣也有清淡，保留并只推荐其不辣的招牌菜

## 四、辣度判定标准

| 标签 | 判定 |
| --- | --- |
| 不辣 | 完全不放辣椒，或只放葱花/白胡椒 |
| 微辣 | 少量辣椒油/干辣椒，不吃辣人也能接受 |
| 中辣 | 明显辣，大多数人能吃 |
| 重辣 | 专项辣菜，不吃辣慎点，或要求不辣版本 |

## 五、每家店必含字段

- 店名
- 所在区 + 具体地址
- 人均消费（元）
- 招牌菜 3–5 个（每个带辣度）
- 详细推荐理由（具体菜品/工艺/口感/环境/性价比/本地口碑共识，150-200 字）
- 适合场景（家庭聚餐 / 商务宴请 / 朋友小聚 / 一人食 / 夜宵 / 早餐 / 下午茶）
- 避坑提示（如有）
