## 用户口味偏好声明

本指南在选店环节主动适配了下列偏好：

1. **不太吃辣**：已排除重辣专门店（重庆火锅店、正宗湘菜馆、辣子鸡/水煮鱼等专门店）。
2. **微辣可接受**：部分店铺的招牌菜可能是微辣，会在条目里注明。
3. **武汉本地经典全部保留**：热干面、鸭脖（周黑鸭/精武）、豆皮、烧麦、汤包、糊汤粉、牛肉面、面窝等必留，并在辣度说明里标注"可要求不辣/微辣"。
4. **混合型店铺**：如果一家店既有重辣也有清淡，保留并只推荐其不辣的招牌菜，重辣菜品会标注"此菜辣，可要求不辣版本"。

## 辣度标注规范

每家店的招牌菜都带辣度标签：

- **不辣**：完全不放辣椒，或只放葱花/白胡椒
- **微辣**：有少量辣椒油/干辣椒调味，不吃辣的人也能接受
- **中辣**：明显辣但大多数人能吃
- **重辣**：专项辣菜，不太吃辣的用户慎点，或要求厨房做不辣版本
